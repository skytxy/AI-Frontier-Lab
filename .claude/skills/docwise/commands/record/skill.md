# Docwise: Record

Save lessons learned to improve future iterations.

## Purpose

When you discover a pattern, rule, or best practice, record it so the skill evolves.

## Triggers (shorthand)

- "record this"
- "记下来"
- "内化这个"
- "update paradigm"

## Usage

```
/docwise:record "internal links should not have .md extension"
/docwise:record "添加UTF-8编码检查规范"
```

## Parameters

- `lesson`: The lesson to record

## Feedback Levels

| Level | Target | Example |
|-------|--------|---------|
| 1 | Chapter content | Missing concept explanation |
| 2 | Chapter config | Wrong validation criteria |
| 3 | Paradigm doc | New gap type discovered |
| 4 | Skill code | Agent behavior improvement |

## What to Record

- New gap categories discovered
- Better collaboration patterns
- Quality standards to enforce
- Link format rules
- Encoding issues
